# 0.1.5 — 2026-05-23

## Changed

- **`package.json` scripts**: Migrated all script invocations from `tsx` to `bun run`; `start:stdio` and `start:http` now use `bun ./dist/index.js` directly; `format` flag corrected to `--unsafe` without trailing `.`; removed legacy `start` alias.
- **`package.json` files**: Removed `changelog/` and `AGENTS.md` from the published `files[]` array.
- **`server.json` `runtimeHint`**: Changed from `"node"` to `"bun"` for both stdio and HTTP packages.
- **`Dockerfile` LABEL**: `org.opencontainers.image.description` updated to match canonical tagline.
- **`manifest.json`**: Added `repository`, `homepage`, and `license` fields; expanded `author` and `compatibility` objects from inline to block form.
- **`CLAUDE.md`**: Header renamed "Agent Protocol"; commands table updated to `bun run`; skills table synced (added `tool-defs-analysis`, `api-linter`, `api-telemetry`, `release-and-publish`; updated `maintenance`, `report-issue-*` entries); bundling and publishing sections rewritten.
- **`.gitignore`**: Aligned to gold-standard template (OS, editor, language, and registry sections added).
- **`.dockerignore`**: Added trailing newline.

## Removed

- **`tsx` devDependency**: Removed (scripts now run natively under `bun run`).
