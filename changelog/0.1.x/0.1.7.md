# 0.1.7 — 2026-05-24

## Fixed

- **`depth_km`**: Marked nullable in `EarthquakeEvent`, `EarthquakeEventSchema`, `EarthquakeEventOutput`, `UsgsFeature`, and `EmscFeature` geometry coordinates. Historical seismic events from USGS can have null depth — previously caused a type crash. Format renders null depth as `'unknown'`.
- **`earthquake_get_event` / `earthquake-event` resource descriptions**: Removed false claim that the response includes "products metadata". Only felt reports, ShakeMap intensity, PAGER alert, tsunami flag, and magnitude type are returned.
