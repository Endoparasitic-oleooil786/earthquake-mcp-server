# 0.1.10 — 2026-05-30

## Changed

- **`earthquake_search`** — `totalCount`, `truncated`, and `notice` moved from `output` to a typed `enrichment` block; both `structuredContent` JSON and `content[]` markdown trailer now carry these values. `total_count` (output key) is renamed `totalCount` (enrichment key); value and presence are preserved.
- **`earthquake_get_feed`** — empty-feed recovery guidance added as `enrichment.notice`; surfaced in both `structuredContent` and `content[]` when the feed returns zero events.
- **`@cyanheads/mcp-ts-core`** ^0.9.13 → ^0.9.16
