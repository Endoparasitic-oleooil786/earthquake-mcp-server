Key features include:

## 🔍 Understanding the Data

The information presented through this interface comes from official seismic sensors located worldwide. These sensors detect ground motion and transmit the intensity and location of events to the monitoring centers.

* Magnitude: This number describes the energy released by the earthquake.
* Depth: This indicates how far below the surface the shift occurred.
* Timestamp: This documents the exact moment the seismic wave reached the sensor.
* Location: This maps the event to a specific grid point on Earth.

If you notice a high frequency of alerts, the system is performing its job accurately by reporting all detected activity.

## 🛠️ Frequently Asked Questions

**Does this software record my personal data?**
The tool does not collect or transmit your personal information. It only exchanges messages with official earthquake monitoring servers.

**What should I do if the software loses connection?**
Check your local internet connection first. If your connection status is normal, wait a few minutes and try again. The servers occasionally update their data streams, which may cause temporary delays in communication.

**Can I run multiple searches at once?**
Yes, the application supports multiple requests. It manages these requests in a queue to ensure you receive accurate and organized results.

**How do I uninstall the program?**
Navigate to the Windows Settings menu, select Apps, find the earthquake-mcp-server in the list, and select Uninstall. This removes the program files and the desktop shortcut from your machine.
